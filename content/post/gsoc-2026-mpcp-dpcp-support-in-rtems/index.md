Congratulations to **Ivy Zhuang** (Santa Clara University) for being accepted to **Google Summer of Code 2026** with the RTEMS project!

I will be mentoring Ivy together with Gedare Bloom on a project that brings **MPCP** and **DPCP**—two classical multiprocessor synchronization protocols—into RTEMS. This extends the current set of locking mechanisms and provides developers with more flexibility when designing real-time systems.

The work involves integrating legacy implementations into the modern RTEMS codebase, resolving compatibility issues, and building comprehensive SMP test suites to ensure correctness and maintainability.

I’m particularly excited about this project because it connects real-time systems theory with practical RTOS implementation—something we don’t often see done end-to-end.

Looking forward to a productive and fun summer!
